# Zoepix boilerplate

*Cloning from Git and starting a new project*
- Clone new repo from git, change folder name and make initial commit


*Setting up local WP dev environment with MAMP*
- Follow the steps in this tutorial https://www.youtube.com/watch?v=OriRDlUbAFg&t=47s
- Save details on Trello card
- Access WP backend via http://localhost:8888/zoepix/wp-login.php
- Make sure zoepix > style.css has correct theme details and apply correct theme on WP backend through Appearance > Themes
